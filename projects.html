Car hire management system:
Developed a web-based car rental system using Html, Css and JavaScript as my front end and php and MySQL as backend.
The system allows users to make reservations online .
The system can provide real-time updates on the availability and status of vehicles in the fleet, helping the rental company manage its inventory more efficiently.
It is user friendly a well-designed web interface simplifies the reservation process, making it easy for customers to browse available cars, choose options, and complete bookings.
